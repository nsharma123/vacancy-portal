Job Manager & Career, a quick & easy way to manage your job vacancies on your WordPress website along with an apply now form. 

== Description ==

The Job Manager & Career plugin from ThemeHigh lets you manage the job posts efficiently. You can set up the career feed by simply copy-pasting the shortcode on the respective page. Using this job board plugin add all the details like job title, description, and any other additional features based on your requirement.

As the jobs can be listed according to the locations, job types, and category, it offers extra comfort to both the admin and the user.

= JOB MANAGER & CAREER AT A GLANCE =

= ‚òû Add and Manage Job Post =
The job manager & career plugin provides the space to add and manage the details related to the job like title, description, and job expiry.

= ‚òû List the jobs under locations, job types, and categories =
This career plugin allows you to include the location, type of job, and related category. Therefore, both the job seeker and employer can easily sort them based on these details.

= ‚òû Multiple methods to apply for a job post =
The plugin lets you create an apply form with required fields to collect data from the applicants. If in case you want to receive the application via mail, simply disable the form and provide the corresponding email address.

= ‚òû Feature special jobs with an extra badge =
Feature any urgent jobs with a special badge using the Featured Listing option. It helps to highlight a particular job from others. You can use this feature by simply selecting the Featured Job checkbox while adding or editing a job.

= ‚òû Upload the resumes in multiple formats =
This WordPress job board plugin allows the applicants to upload the resume in multiple formats like .doc, pdf, .docx.

= ‚òû Display of expired or filled jobs =
This option allows the admin to hide or show the expired and filled jobs. It helps the applicants get a better view of the vacancies and job listings.

= ‚òû Undo all the changes in a single click =
The career plugin provides a Reset To Default button in the General settings. By clicking on this button, you can undo all the changes that you have made and reset to the original.

= ‚òû Add additional features for the job =
Apart from adding the job title, and description, you can also include some additional features related to the job. There may come many situations to add extra features related to the job such as experience, salary, and so on.

= ‚òû The feature to import the settings data =
You can import the plugin data by simply copying the text from the Export textbox and replacing the data in the textbox of another install.

== Installation ==
= Minimum Requirements =
* WordPress 4.7 or greater
* PHP version 5.2.4 or greater (PHP 5.6 or greater is recommended)
* MySQL version 5.0 or greater (MySQL 5.6 or greater is recommended)

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'Themehigh Job Manager' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
Manual installation method requires downloading the 'Themehigh Job Manager' plugin and uploading it to your web server via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
